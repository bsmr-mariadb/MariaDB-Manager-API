User Management via API

Get all Users

The HTTP method to be used is:
URI will be /user
Parameters: fields fields
Comma separated list of the field names that you would like to be returned. If not specified, all fields are returned.
Default: All fields



Add or Update a User

The HTTP method to be used is:
URI will be /user/{username}
Parameters: name, name
The ordinary name of the user (URL encoded string).   password password
Password for the user (URL encoded string).





Delete a User

The HTTP method to be used is:
URI will be /user/{username}
No parameters



Get User Information

The HTTP method to be used is:
URI will be /user/{username}
Parameters: fields fields
Comma separated list of the field names that you would like to be returned. If not specified, all fields are returned.
Default: All fields




Authenticate User

The HTTP method to be used is:
URI will be /user/{username}
Parameter is a POST data item, called password password
The password belonging to the user being authenticated (URL encoded string).




Add or Update a User Property

The HTTP method to be used is:
URI will be /user/{username}/property/{propertyname}
Parameters: value value
The value for the given property (URL encoded string).





Get a User Property

The HTTP method to be used is:
URI will be /user/{username}/property/{propertyname}
Parameters: none




Delete a User Property

The HTTP method to be used is:
URI will be /user/{username}/property/{propertyname}
No parameters




Add a User Tag

The HTTP method to be used is:
URI will be /user/{username}/monitortag/{tagname}
Parameters: systemid, nodeid, monitor
Note: nodeid can be zero for a system monitor







Get Named User Tags

The HTTP method to be used is:
URI will be /user/{username}/monitortag/{tagname}
Parameters: None




Get All User Tags

The HTTP method to be used is:
URI will be /user/{username}/monitortag
Parameters: None



Delete Named User Tags

The HTTP method to be used is:
URI will be /user/{username}/monitortag/{tagname}
Parameters: None




Delete All User Tags

The HTTP method to be used is:
