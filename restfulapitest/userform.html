User Management via SCDS API

Get all Users

The HTTP method to be used is:
URI will be /user
Parameters: fields



Add or Update a User

The HTTP method to be used is:
URI will be /user/{username}
When PUTting, parameters are passed as JSON (name and password)





Delete a User

The HTTP method to be used is:
URI will be /user/{username}
No parameters



Get User Information

The HTTP method to be used is:
URI will be /user/{username}
No parameters



Authenticate User

The HTTP method to be used is:
URI will be /user/{username}
Parameter is a POST data item, called password




Add or Update a User Property

The HTTP method to be used is:
URI will be /user/{username}/property/{propertyname}
When PUTting, parameter is passed as JSON (value)





Delete a User Property

The HTTP method to be used is:
URI will be /user/{username}/property/{propertyname}
No parameters
