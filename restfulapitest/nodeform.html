Node Management via SkySQL Manager API

Get All Node States

The HTTP method to be used is:
URI will be /nodestate
Parameters: fields fields
Comma separated list of the field names that you would like to be returned. If not specified, all fields are returned.
Default: All fields



Get Node States for System Type

The HTTP method to be used is:
URI will be /nodestate/{systemtype}
Parameters: fields fields
Comma separated list of the field names that you would like to be returned. If not specified, all fields are returned.
Default: All fields




Get all Nodes for a System

The HTTP method to be used is:
URI will be /system/{systemid}/node
Parameters: state, state
Optional - limit results to nodes that are in the specified state.   fields fields
Comma separated list of the field names that you would like to be returned. If not specified, all fields are returned.
Default: All fields





Create or Update a Node

The HTTP method to be used is:
URI will be /system/{systemid}/node/{nodeid}
Parameters: name, name
The name you want to give to the node (text).    state, state
The state code to be given to the node, must be selected from valid node states (string).   stateid, stateid
The state ID number (integer) that refers to the state to be given to the node. It will only be used if state is not provided. The ID will be used to look up the state in the API internal tables, and the number given must refer to a valid node state, or it will be ignored.   hostname, hostname
The hostname for the node (text).   publicip, publicip
The public IP address of the node in the usual form, e.g. 12.34.56.78   privateip, privateip
The private IP address of the node in the usual form, e.g. 23.45.67.89   port, port
The port number used to access the database on this node - empty or zero if the default is to be used.   instanceid, instanceid
The ID number for the instance (integer).   username, username
The user name for the node (text).   passwd passwd
The password for the node (text).
