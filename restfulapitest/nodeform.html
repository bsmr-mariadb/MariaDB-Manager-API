Node Management via SCDS API

Get all Node States

The HTTP method to be used is:
URI will be /nodestate
Parameters: fields
Default: All fields



Get Selected Node State(s)

The HTTP method to be used is:
URI will be /nodestate/{idordescription}
Parameters: fields
Default: All fields




Get all Nodes

The HTTP method to be used is:
URI will be /system/{systemid}/node
Parameters: fields




Create or Update a Node

The HTTP method to be used is:
URI will be /system/{systemid}/node/{nodeid}
Parameters name, status, hostname, publicip, privateip, instanceid, username, passwd












Delete a Node

The HTTP method to be used is:
URI will be /system/{systemid}/node/{nodeid}
No parameters




Get Node Information

The HTTP method to be used is:
URI will be /system/{systemid}/node/{nodeid}
Parameters: fields
