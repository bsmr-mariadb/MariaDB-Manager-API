Node Management via SkySQL Manager API

Get all Node States

The HTTP method to be used is:
URI will be /nodestate
Parameters: fields fields
Comma separated list of the field names that you would like to be returned. If not specified, all fields are returned.
Default: All fields



Get Selected Node State(s)

The HTTP method to be used is:
URI will be /nodestate/{idordescription}
Parameters: fields fields
Comma separated list of the field names that you would like to be returned. If not specified, all fields are returned.
Default: All fields




Get all Nodes

The HTTP method to be used is:
URI will be /system/{systemid}/node
Parameters: fields fields
Comma separated list of the field names that you would like to be returned. If not specified, all fields are returned.
Default: All fields




Create or Update a Node

The HTTP method to be used is:
URI will be /system/{systemid}/node/{nodeid}
Parameters: name, name
The name you want to give to the node (text).   state, state
The state ID to be given to the system (integer).   hostname, hostname
The hostname for the node (text).   publicip, publicip
The public IP address of the node in the usual form, e.g. 12.34.56.78   privateip, privateip
The private IP address of the node in the usual form, e.g. 23.45.67.89   instanceid, instanceid
The ID number for the instance (integer).   username, username
The user name for the node (text).   passwd passwd
The password for the node (text).
