Backup Management via SCDS API

Get all Backup States

The HTTP method to be used is:
URI will be /backupstates
No parameters


Get System Backups

The HTTP method to be used is:
URI will be /system/{systemid}/backup
Parameters: from, to, fields, limit, offset
Defaults are forever, forever, all fields, 10, 0 respectively








Update a System Backup

The HTTP method to be used is:
URI will be /system/{systemid}/backup/{backupid}
Parameters: state, size, storage, binlog, log
